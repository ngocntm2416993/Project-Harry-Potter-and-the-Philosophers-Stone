
# ⚡ Harry Potter and the Philosopher's Stone (OOP Project)

Welcome to our game project inspired by the magical world of Harry Potter and the Philosopher's Stone! This game was developed as a hands-on project for our Object-Oriented Programming (OOP) course.

Our team has brought the magic of Harry's first year at Hogwarts to life, translating iconic moments into interactive gameplay. While staying true to the spirit of the original book and movie, we have introduced several custom tweaks, gameplay mechanics, and level adjustments to ensure a balanced, engaging, and fun user experience.


## 🎮 Key Features & OOP Concepts
Dynamic Levels: Experience unique stages inspired by the story, each optimized for specific gameplay challenges.

OOP Architecture: Built from the ground up using core OOP principles such as Inheritance, Polymorphism, Encapsulation, and Abstraction to manage game entities.
## 🛠️ Development Tools & Technologies

`Java & JDK 26`: main programming language

`Java Swing & JavaFX`: Java libraries

`Piskel & Ibis`: creating pixel art assets, character sprites
## 🚀 Installation & Setup
### Step 1: Download the Game
* [Download Game via Google Drive](https://drive.google.com/drive/folders/1Wg7aA1SWRLFbZaAh5ZXlGLQQ3V4vqERk?usp=drive_link)

### Step 2: Open Command Prompt (cmd)
* Press the `Win + R` keys, type `cmd`, and hit **Enter** to open the Command Prompt (or you can type `cmd` on Search bar)

### Step 3: Navigate to the Game Folder
Use the Windows **"Copy as path"** feature to jump into the game directory instantly:
1. Hold the `Shift` key and **Right-Click** on your extracted game folder.
2. Click **"Copy as path"**.
3. In your `cmd` window, type `cd /d ` and paste your path (press `Ctrl + V` or Right-Click to paste). 

```cd/d path/to/your/extracted-folder```
